# 🚛 Kalifrance CRM

Outil de gestion de prospection commerciale pour l'équipe Kalifrance Déménagement.

## 🔗 Accès
👉 [Ouvrir le CRM](https://quincy-hsr.github.io/kalifrance-crm)

## 📋 Fonctionnalités

- 📊 **Dashboard** — Vue d'ensemble de la prospection en temps réel
- 📅 **Agenda** — Planification des tournées de visites
- 🏢 **Agences prospects** — Suivi de toutes les agences démarchées
- 🤝 **Agences partenaires** — Suivi des contrats signés
- 📞 **Relances** — Planification automatique des relances
- 📝 **Bilan journée** — Résultats de visite par agence

## 👥 Rôles

| Rôle | Accès |
|------|-------|
| 👑 Admin | Accès total |
| 👩‍💼 Planificatrice (Annah) | Planifie les tournées |
| 🚶 Commercial (Quincy) | Renseigne les résultats |

## ⚙️ Technologies

- HTML / CSS / JavaScript (vanilla)
- [Supabase](https://supabase.com) — Base de données temps réel
- [OpenStreetMap](https://openstreetmap.org) — Suggestions d'adresses
- Hébergé sur [GitHub Pages](https://pages.github.com)
